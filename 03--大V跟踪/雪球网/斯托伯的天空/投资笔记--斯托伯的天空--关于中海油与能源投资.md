## 一、中海油H投资记录

### 1.1 投资收益

|------|------|
| **持有时间** | 三年 |
| **初始投入收益** | **三倍** |

| **作者评价** | "一年三倍易，三年三倍难，慢就是快啊" |

### 1.2 为什么买H股？

**核心逻辑**：
> "优质股在港股肯定比在A股好，不管是紫金矿业、中海油还是招行。港股更能充分体现对基本面的反应，哪怕它流动性差了点。"

**港股优势**：
- 更能充分体现对基本面的反应
- 不容易站岗
- 新股不会把未来三五年预期打满
- 更适合长期投资

### 1.3 中海油的竞争优势

**管理能力和业务能力**：
- 这几年大幅度提升
- 渤海油田等项目做得很好

**盈利持续性**：
- 油价估值中枢提升
- 盈利持续性好起来了

**抗跌能力**：
> "中海油硬是抗着油价下跌而股价震荡，确实强。"

## 二、能源板块投资逻辑

### 2.1 油价与能源股

**怀念当年**：
> "相比较而言，煤炭原油里面的资金要老道稳重得非常多，怀念当年持有陕煤和中海油的日子。"

### 2.2 能源安全

- 能源安全重要性越来越大
- 电力设施建设是重点方向
- 欧美电力设施已四五十年，需要大规模改造

### 2.3 能源股特征

| 特征 | 说明 |
|------|------|
| **资金属性** | 老道、稳重 |
| **波动性** | 相对较小 |
| **分红** | 高股息 |
| **长期逻辑** | 能源安全、基础设施投资 |

## 三、陕西煤业

### 3.1 投资操作

- 建立观察仓

### 3.2 煤炭投资逻辑

- 煤价本轮库存周期和政策周期的底部已找到

## 四、能源与电力的关系

### 4.1 电力需求增长

**驱动因素**：
- 国内67万亿投资建设，电力设施是重点
- 欧美电力设施老化（四五十年），需要大规模改造升级
- 电动车普及
- 机器人等新需求

### 4.2 能源与大宗的关系

> "未来能源是电力。这次国内已公布的67万亿投资建设，电力设施也是重点方向。"
