## 一、紫金矿业

### 1.1 核心投资逻辑

| 优势 | 说明 |
|------|------|
| **超级周期** | 在一个超级长的大宗商品牛市上 |
| **复合体** | 超级铜金复合体 |
| **管理能力** | 非常优秀的管理能力 |
| **估值** | 目前市场给的估值偏低 |

### 1.2 大胆预测

> "紫金矿业市值超过茅台，那不是拍屁股瞎说，那是有依据的。"

**现状**：
- 茅台市值已被几家企业陆续超过
- 未来还有更多企业超过茅台
- 紫金是其中之一

### 1.3 历史操作

- **2019-2020年**：策略里已有黄金，包括紫金
- **朴素逻辑**：对货币问题带来的不确定性倒逼的一种配置
- **当时主要仓位**：大消费（白酒、免税）

### 1.4 与白酒的对比

- 作者曾在白酒最高位逃顶（茅台评估领先市场）
- 2021年下半年，黄金逐渐成为重点研究对象
- 时代变化，矿业可能成为好生意

### 1.5 与赤峰黄金对比

- **期货与股票同步见顶**：2020年8月7日
- **提前启动**：期货2018年下半年启动，股票2019年下半年启动
- **市场先生**：有时候是好人，虽然他不说一句话

## 二、云铝股份（云妹）

### 2.1 昵称由来

作者亲切称之为"云妹"、"笨鸟"

### 2.2 投资表现

- **单骑救主**：保证账户红盘
- **第一大仓位**：再次跃升第一大仓位
- **不知不觉到目标价**：稳扎稳打

### 2.3 投资逻辑

**与黄金相似的因素**：
1. 美元信用下降 + 逆全球化
2. 各国注重保护本国货币主权信用
3. 资源国谨慎保持大宗商品输出
4. 全球定价重置 → 长期高位和持续上行

**铝的特有逻辑**：
- 电解铝产能受限（国内产能天花板）
- 铝土矿分布不均（中国大量依赖进口）
- 电力需求爆发（电动车、机器人、电网改造）

### 2.4 弹性特征

> "铝股的弹性，几乎是大宗商品股票里面最强。"

**历史表现**：
- 2005-2007年：5-8倍涨幅
- 美国铝业：曾涨约20倍
- 中国铝业：涨4倍多

### 2.5 成本结构

| 成本项 | 占比 |
|--------|------|
| 氧化铝 | 30%-40% |
| 电力 | 30%-40% |
| 阳极价格 | 10%-15% |
| 其他 | 剩余 |

### 2.6 目标价

- 短期目标价已到
- 后续可期待季度常态利润到25亿

## 三、赤峰黄金

### 3.1 半年报分析

**业绩不达标**：
- 技改和矿山品味问题存在
- 影响性权重看不出来
- 下调全年产能目标：16.7吨→16吨

**风险点**：
- 铜矿业务大亏（半年报未解释）
- 稀土产品亏损
- 募集资金花不出去

### 3.2 筹码结构

**变差**：
- 社保基金大量减仓
- 散户大量增加
- 最近两个月走得很差的直接原因

**内幕交易嫌疑**：
- 关联资金提前拿到信息跑路（六月份）
- 大A投资的风险

### 3.3 亮点

- 锦泰探转采（刚买就赚钱）
- 塞班增储70多吨（储量大幅提升）
- 全年动态估值在AH股偏低

## 四、中国铝业

### 4.1 氧化铝价格

- 山西地区限产政策延续
- 电解铝产能复产进度慢于预期
- 供需缺口支撑铝价

### 4.2 绿色能源转型

- 云南水电铝产能占比提升至35%
- 碳成本优势显现

## 八、个股选择原则

### 8.1 选股标准

1. **长景气度**：行业有长期逻辑
2. **能看透**：理解业务模式
3. **低估值**：有安全边际
4. **好管理**：管理层优秀
5. **分红好**：现金流稳定

### 8.2 减仓原则

- 短期情绪过高
- 股价涨幅过大（如150%以上）
- 估值修复完成
