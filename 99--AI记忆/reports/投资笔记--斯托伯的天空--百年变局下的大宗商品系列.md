## 一、核心投资框架：百年变局论

### 1.1 "百年变局"的核心含义

作者反复强调的"百年变局"是指：
- **40年稳定期结束**：和平发展被看作是永恒的时代已经过去
- **逆全球化**：各国开始注重保护本国货币主权信用
- **资源国谨慎输出**：资源输出国开始谨慎保持大宗商品输出
- **全球定价重置**：大宗商品长期高位和持续上行的动能

### 1.2 与历史周期的对比

| 周期 | 时间 | 驱动因素 | 代表事件 |
|------|------|----------|----------|
| **上一轮黄金牛市** | 2000-2012年 | 地缘政治、经济复苏、通胀 | 911、金融危机、欧债危机 |
| **本轮黄金牛市** | 2016年至今 | 百年变局、美元信用下降 | 逆全球化、战争频发 |

作者亲历2000-2012年黄金牛市，900多美元建仓，1500多美元平仓。

## 二、黄金篇：货币之王的回归

### 2.1 黄金上涨的核心逻辑

**主逻辑：美元信用下降**
- 美元信用下降导致全球主权货币贬值
- 黄金相对于所有主权货币升值
- 不仅仅是美元贬值，而是全球货币体系的重构

### 2.2 黄金与美元的关系

> "黄金的上涨并不必然导致美元的贬值，美元的购买力目前主要是跟其他货币去比较而不是黄金。"

> "黄金的上涨，导致的是全球所有主权货币的贬值，美元只是其中之一，相对而言还算贬得最少的。"

### 2.3 投资观点

- **黄金期货牛市启动时间**：2018年下半年
- **黄金股启动时间**：2019年下半年（以赤峰黄金为例）
- **同步见顶**：2020年8月7日

### 2.4 对战争与黄金的看法

> "如果把俄乌战争看成金价重要参考的，基本走偏了。"

- 战争不是黄金上涨的主因
- 真正的逻辑是百年变局下的货币体系重构
- 特朗普上台时如果是因为战争，那黄金就该跌了

## 三、铜篇：铜博士的超级大周期

### 3.1 为什么叫"铜博士"

铜是经济活动中需要最多的金属之一：
- **铜价上涨** → 预示经济好转
- **铜价下跌** → 预示经济衰退
- 被誉为"全球经济的水银柱"

### 3.2 铜的核心属性

| 属性 | 说明 |
|------|------|
| **导电性** | 仅次于银，电力行业主要用途 |
| **导热性** | 优良，用于散热设备 |
| **延展性** | 柔软易加工 |
| **化学稳定性** | 常温下不与干燥空气反应 |

**铜的消费结构**：电力行业是主要用途，决定铜消耗量的主要是电力行业需求。

### 3.3 铜的超级周期逻辑

#### 供给端：越挖越少，越来越稀缺

- **产能高度集中**：智利占全球产量28%
- **主力矿山老化**：排名前20的矿山，70%是30年前发现的
- **找矿困难**：2014年卡库拉铜矿发现后，10年无特大铜矿发现
- **开发周期长**：从勘探到出产能需要6-8年

#### 价格中枢抬升

铜价历史走势：
- 1000多 → 4000多 → 5000多 → 现在7000多美元/吨
- 呈周期性向上波动

### 3.4 投资结论

- **仅次于黄金的第二大大宗商品投资机会**
- **百年大变局下超级周期的受益资产**
- 铜价要涨到"惊天动地"，20000美元可能都打不住

## 四、铝篇：中国的铝，世界的铝

### 4.1 铝与黄金的相似因素

1. **美元信用下降** + **逆全球化**
2. 各国注重保护本国货币主权信用
3. 资源国谨慎保持大宗商品输出
4. **全球定价重置** → 长期高位和持续上行

### 4.2 铝的特有逻辑

#### 供给端：产能限制

- **电解铝产能受限**：国内产能天花板明确
- **铝土矿分布不均**：
  - 中国、日韩大量依赖进口
  - 大储量高品质矿在澳大利亚、几内亚
  - 制造业大国vs资源国的矛盾

#### 成本结构

| 成本项 | 占比 |
|--------|------|
| 氧化铝 | 30%-40% |
| 电力 | 30%-40% |
| 阳极价格 | 10%-15% |
| 人工等其他 | 剩余 |

### 4.3 铝的弹性

> "铝股的弹性，几乎是大宗商品股票里面最强。"

历史表现：
- 2005-2007年铝股大涨：5-8倍涨幅
- 美国铝业：曾涨约20倍
- 中国铝业：涨4倍多

### 4.4 需求端逻辑

- **电力设施建设**：国内67万亿投资，电力设施是重点
- **欧美电网改造**：设施已四五十年，需要大规模升级
- **电动车普及**、**机器人**等新需求

## 五、投资方法论

### 5.1 交易纪律

- **-7%建仓法**：当日跌幅达7%时挂单买入
- **不追跌**：下跌趋势的票不出手，除非明显低估
- **击球区**：价格到了舒适区间才买入

### 5.2 选股偏好

| 类型 | 偏好 |
|------|------|
| **行业** | 资源股、周期股 |
| **逻辑** | 长景气度、能看透的行业 |
| **估值** | 低估值、分红好 |
| **心态** | 拿得住，相信"钱最终都要去能赚钱的地方" |

### 5.3 配置策略

- **紫金矿业**（铜+金）
- **中海油**（能源）
- **云铝股份**（铝）
- **神火股份**（铝+煤）
