# 产业链分析框架 (Industry Chain Analysis Framework)

> 用于分析目标股票在产业链中的位置、议价能力，以及发现上下游更好的投资机会
> 创建时间：2026-03-11

---

## 一、分析目标

1. **评估目标公司在产业链中的议价能力**
2. **识别上游/下游/横向的投资机会**
3. **判断行业竞争格局和护城河**
4. **发现产业链风险传导路径**

---

## 二、产业链四维度分析

### 2.1 上游分析（供给侧）

#### 核心问题

| 检查项 | 关键问题 | 数据/信息来源 |
|--------|----------|--------------|
| **供应商集中度** | 前五大供应商占比？是否有单一供应商依赖？ | 年报"前五大供应商"、招股书 |
| **原材料价格趋势** | 核心原材料涨价还是跌价？ | 行业协会、大宗商品价格 |
| **供应稳定性** | 是否有断供风险？是否依赖进口？ | 地缘政治、贸易政策 |
| **替代可行性** | 切换供应商的难度和成本？ | 技术壁垒、认证周期 |
| **议价能力** | 公司对上游有议价权吗？ | 应付账款周转天数、预付款比例 |

#### 评估标准

| 上游状态 | 特征 | 对公司的影响 | 投资建议 |
|----------|------|-------------|----------|
| **强势上游** | 集中度高、不可替代 | 成本高、利润薄 | 避开或寻找上游标的 |
| **弱势上游** | 分散、标准化产品 | 成本低、利润厚 | 利好，可投资 |
| **波动上游** | 价格周期性波动大 | 毛利率不稳定 | 关注周期位置 |

#### 实操案例

**阳光电源（逆变器/储能）上游：**
- IGBT芯片：英飞凌、三菱等进口依赖（⚠️ 卡脖子风险）
- 碳酸锂：国内锂矿企业供应（价格波动大）
- 结论：上游议价能力弱，成本压力大

**招商银行上游（资金来源）：**
- 存款人：高度分散（✅ 议价能力强）
- 同业拆借：市场化定价
- 结论：上游资金成本可控，当前利率下行利好

---

### 2.2 下游分析（需求侧）

#### 核心问题

| 检查项 | 关键问题 | 数据/信息来源 |
|--------|----------|--------------|
| **客户集中度** | 前五大客户占比？大客户依赖风险？ | 年报"前五大客户" |
| **需求增长** | 下游行业景气度？需求增速？ | 行业报告、宏观数据 |
| **客户议价能力** | 客户是否强势？账期长短？ | 应收账款周转天数 |
| **需求稳定性** | 是周期性需求还是刚性需求？ | 行业属性 |
| **替代威胁** | 客户是否容易转向竞争对手？ | 产品差异化、转换成本 |

#### 评估标准

| 下游状态 | 特征 | 对公司的影响 | 投资建议 |
|----------|------|-------------|----------|
| **强势下游** | 集中度高、账期长 | 应收账款高、现金流差 | 谨慎 |
| **弱势下游** | 分散、预付款 | 现金流好、议价能力强 | 利好 |
| **增长下游** | 需求快速增长 | 订单增长、产能扩张 | 积极 |
| **衰退下游** | 需求萎缩 | 产能过剩、价格战 | 避开 |

#### 实操案例

**阳光电源下游：**
- 光伏电站运营商：集中度中等，账期较长（⚠️ 应收账款高）
- 储能系统集成商：需求爆发但价格战激烈（⚠️ 毛利率承压）
- 结论：下游需求增长但客户议价能力增强

**招商银行下游：**
- 零售客户：高度分散（✅ 抗风险）
- 对公客户：房地产敞口4.37%（⚠️ 需关注）
- 结论：零售基因提供安全边际

### 2.3 竞争格局分析

#### 核心问题

| 检查项 | 关键问题 | 分析维度 |
|--------|----------|----------|
| **行业集中度** | CR3/CR5是多少？龙头是谁？ | 市场份额、行业研报 |
| **进入壁垒** | 新进入者容易吗？ | 技术、资金、牌照、品牌 |
| **替代品威胁** | 有其他技术路线替代吗？ | 技术变革风险 |
| **同业竞争** | 价格战激烈吗？差异化程度？ | 毛利率趋势、行业盈利能力 |

#### 波特五力模型简化版

```
        上游供应商议价能力
                ↑
新进入者威胁 ←  [公司]  → 下游客户议价能力
                ↓
        同业竞争 + 替代品威胁
```

#### 评估标准

| 竞争格局 | 特征 | 对公司的影响 | 投资策略 |
|----------|------|-------------|----------|
| **寡头垄断** | CR3>50%，壁垒高 | 龙头盈利稳定 | 买龙头 |
| **垄断竞争** | 差异化竞争 | 有护城河的企业胜出 | 选龙头或细分冠军 |
| **完全竞争** | 同质化、价格战 | 利润薄、波动大 | 避开或选成本最低者 |
| **新兴行业** | 格局未定、快速变化 | 高风险高回报 | 分散投资或观望 |

#### 实操案例

**光伏逆变器行业：**
- CR2（华为+阳光）>50%，格局稳定（✅）
- 但储能行业壁垒低、价格战激烈（⚠️）
- 结论：逆变器是寡头垄断，储能是完全竞争

**银行业：**
- 国有行+股份行寡头垄断（✅）
- 但互联网金融冲击（⚠️）
- 结论：传统银行护城河仍稳固

### 2.4 产业链机会挖掘

#### 三个关键问题

分析完产业链后，必须问自己：

**问题1：上游有没有更好的标的？**
```
分析路径：
目标公司毛利率承压 → 上游原材料涨价 → 上游公司受益
→ 投资上游 instead of 下游

例：阳光电源（下游）vs 碳酸锂矿（上游）
- 阳光电源受碳酸锂价格波动影响大
- 碳酸锂涨价时，矿企直接受益
- 结论：根据周期位置选择上游或下游
```

**问题2：下游有没有更好的标的？**
```
分析路径：
目标公司订单增长 → 下游需求爆发 → 下游景气度高
→ 直接投资下游应用端

例：宁德时代（中游电池）vs 电动车企（下游）
- 电池竞争激烈，车企差异化竞争
- 若看好电动车，直接投车企可能更好
- 结论：比较产业链各环节估值和成长性
```

**问题3：横向有没有更好的标的？**
```
分析路径：
同行业比较 → 估值、成长性、资产质量
→ 选性价比最高的

例：银行股比较
- 招行：PB 0.93，ROE 14.5%，不良率0.95%
- 平安银行：PB更低但地产风险高
- 兴业银行：对公占比高，周期性强
- 结论：招行是横向最优选择
```

## 三、产业链风险传导

### 3.1 上游风险传导路径

```
上游原材料涨价
    ↓
中游制造成本上升
    ↓
毛利率压缩（若无法转嫁）
    ↓
或：产品涨价 → 下游需求减少
    ↓
业绩下滑
```

**案例：**
- 碳酸锂涨价 → 电池成本上升 → 储能系统毛利率下降
- 芯片短缺 → 逆变器产能受限 → 出货量不及预期

### 3.2 下游风险传导路径

```
下游行业衰退/政策变化
    ↓
订单减少/应收账款坏账
    ↓
收入下降/资产减值
    ↓
业绩暴雷
```

**案例：**
- 房地产下行 → 银行对公贷款不良率上升
- 光伏补贴退坡 → 电站投资减少 → 逆变器需求下降

### 3.3 风险评估矩阵

| 风险类型 | 概率 | 影响 | 应对策略 |
|----------|------|------|----------|
| 上游涨价 | 中 | 高 | 关注原材料周期，或投资上游对冲 |
| 下游衰退 | 低 | 高 | 分散下游行业，选刚需行业 |
| 竞争加剧 | 高 | 中 | 选护城河深、成本低的龙头 |
| 技术替代 | 低 | 高 | 关注技术变革，避开夕阳技术 |

## 四、实操流程

### 4.1 快速产业链分析（10分钟）

```
Step 1: 画出产业链图谱（上中下游）
Step 2: 标记目标公司位置
Step 3: 评估上下游议价能力（强/中/弱）
Step 4: 判断竞争格局（寡头/垄断竞争/完全竞争）
Step 5: 回答三个问题：
        - 上游有没有更好的标的？
        - 下游有没有更好的标的？
        - 横向有没有更好的标的？
Step 6: 输出结论
```

### 4.2 深度产业链分析（30分钟）

增加：
- 上下游上市公司梳理（标的挖掘）
- 产业链估值对比（PE/PB分位）
- 产业链景气度判断（周期位置）
- 政策对产业链的影响

## 五、案例库

### 案例1：光伏产业链

```
上游：硅料（通威股份）、硅片（隆基绿能）
中游：电池片、组件（晶科、晶澳）、逆变器（阳光电源）
下游：电站运营商（三峡能源、华能国际）

分析：
- 上游硅料：周期性强，2021-2022年涨价受益，2023-2024年跌价受损
- 中游组件：价格战激烈，同质化竞争
- 下游电站：重资产，受益电价上涨
- 结论：根据周期位置选择上游或下游，中游长期承压
```

### 案例2：银行产业链

```
上游：存款人（资金成本）、央行（政策利率）
中游：各类银行（国有行、股份行、城商行）
下游：零售客户、对公客户（企业、政府）

分析：
- 上游：存款利率下行利好银行息差
- 中游：零售银行（招行）>对公银行（兴业）>地产依赖银行（平安）
- 下游：零售客户抗周期，对公客户受经济影响大
- 结论：选零售基因强、资产质量优的银行
```

### 案例3：新能源车产业链

```
上游：锂矿（赣锋、天齐）、钴镍（华友钴业）
中游：电池（宁德时代、比亚迪）、电机电控
下游：整车（比亚迪、特斯拉、蔚来）、充电桩

分析：
- 上游：受益电动车渗透率提升，但价格波动大
- 中游：电池竞争激烈，产能过剩
- 下游：整车差异化竞争，品牌为王
- 结论：上游看周期位置，中游选成本领先者，下游选品牌强者
```


| 原有框架 | 产业链分析补充 | 整体价值 |
