# 财报分析评分框架 v1.0

> 优秀公司财报的黄金标准与评分体系
> 创建时间：2026-03-11

---

## 一、优秀财报的核心标准（巴菲特原则）

### 1.1 黄金指标：ROE（净资产收益率）

**巴菲特标准**：
- **ROE ≥ 15%**（连续3年以上）→ 优秀公司 ✅
- **ROE ≥ 20%**（连续5年以上）→ 卓越公司 ⭐
- **ROE ≥ 30%** → 凤毛麟角 💎

**ROE 三因素拆解**（杜邦分析）：
```
ROE = 净利率 × 资产周转率 × 权益乘数

高ROE的三种模式：
1. 高净利率模式：茅台（净利率50%+）
2. 高周转模式：沃尔玛（周转率极高）
3. 高杠杆模式：银行（杠杆10-20倍）
```

**⚠️ 注意**：高杠杆带来的高ROE风险较大，优先选择高净利率或高周转模式。

### 1.2 现金流铁律

| 指标 | 优秀标准 | 危险信号 |
|------|---------|---------|
| **经营现金流/净利润** | > 1（长期） | < 0.8（持续）|
| **收现比**（销售收现/营收）| > 1.13（含税） | < 0.9 |
| **自由现金流** | 持续为正 | 长期为负 |

**核心原则**：
> "利润是纸面上的，现金流才是真金白银。"

**危险信号**：利润增长但经营现金流持续为负 → 可能财务造假 ⚠️

### 1.3 盈利能力指标

| 指标 | 优秀 | 良好 | 一般 | 危险 |
|------|------|------|------|------|
| **毛利率** | >40% | 25-40% | 15-25% | <15% |
| **净利率** | >20% | 10-20% | 5-10% | <5% |
| **ROE** | >20% | 15-20% | 10-15% | <10% |
| **ROIC** | >15% | 10-15% | 8-10% | <8% |

**说明**：
- **毛利率**反映产品竞争力（茅台91%，苹果45%）
- **净利率**反映费用控制能力
- **ROIC**（投入资本回报率）比ROE更真实（不受杠杆影响）

## 二、五维财报分析框架

### 2.1 盈利能力（权重25%）

**核心指标**：
1. **毛利率** = (营收-营业成本)/营收
   - 优秀：>40%，良好：25-40%
   - 趋势：稳定或上升 ✅，持续下降 ⚠️

2. **净利率** = 净利润/营收
   - 优秀：>20%，良好：10-20%
   - 关注扣非净利润（剔除非经常性损益）

3. **ROE** = 净利润/净资产
   - 优秀：>15%（连续3年）
   - 拆解：净利率×周转率×杠杆

4. **ROIC** = 息前税后利润/投入资本
   - 优秀：>10%，卓越：>15%

**分析要点**：
- 与同行业对比（横向）
- 与历史数据对比（纵向）
- 关注扣非净利润真实性

### 2.2 偿债能力（权重20%）

**短期偿债**：
| 指标 | 优秀 | 安全 | 危险 |
|------|------|------|------|
| **流动比率** | >2 | 1.5-2 | <1 |
| **速动比率** | >1.5 | 1-1.5 | <0.8 |

**长期偿债**：
| 指标 | 优秀 | 安全 | 危险 |
|------|------|------|------|
| **资产负债率** | <40% | 40-60% | >70% |
| **有息负债率** | <20% | 20-40% | >50% |
| **利息保障倍数** | >5 | 3-5 | <3 |

**⚠️ 危险信号**：
- 存贷双高（货币资金高+有息负债高）→ 可能造假
- 流动比率<1 → 短期偿债压力大
- 资产负债率>70% → 高杠杆风险

### 2.3 营运能力（权重15%）

| 指标 | 优秀 | 良好 | 一般 |
|------|------|------|------|
| **应收账款周转率** | >10次/年 | 5-10次 | <5次 |
| **存货周转率** | 行业前列 | 行业中位 | 低于行业 |
| **总资产周转率** | >1 | 0.5-1 | <0.5 |
| **营业周期** | <60天 | 60-120天 | >120天 |

**分析要点**：
- 周转率越高，资金使用效率越高
- 与同行业对比（零售业周转快，制造业周转慢）
- 应收账款增速 > 营收增速 → 可能放宽信用政策虚增收入 ⚠️

### 2.4 成长能力（权重20%）

| 指标 | 成长股 | 稳定股 | 衰退股 |
|------|--------|--------|--------|
| **营收增长率** | >20% | 10-20% | <5% |
| **净利润增长率** | >25% | 10-25% | <5% |
| **ROE增长率** | 稳定或上升 | 波动小 | 持续下降 |

**质量判断**：
- 内生增长（ organic growth）> 并购增长 ✅
- 主业增长 > 非经常性损益增长 ✅
- 有现金流支撑的增长 > 纯利润增长 ✅

### 2.5 现金流健康度（权重20%）

| 指标 | 优秀 | 良好 | 危险 |
|------|------|------|------|
| **经营现金流净额** | >净利润 | 净利润的80% | <净利润的50% |
| **自由现金流** | 持续为正 | 偶尔为负 | 长期为负 |
| **现金转换周期** | <0（负现金流） | 0-30天 | >60天 |

**三种现金流组合分析**：
```
经营流  投资流   筹资流   企业状态
─────────────────────────────────────
+       +        +        抽血型（危险）
+       +        -        成熟期（健康）✅
+       -        +        成长期（扩张）✅
+       -        -        奶牛型（最佳）⭐
-       +        +        衰退型（危险）⚠️
-       +        -        卖资产度日（危险）
-       -        +        烧钱型（需谨慎）
-       -        -        濒死型（远离）❌
```

## 三、财务造假排雷清单（一票否决项）

### 3.1 审计意见异常 🚨

| 审计意见 | 风险等级 | 处理建议 |
|---------|---------|---------|
| **标准无保留** | 🟢 安全 | 正常分析 |
| **保留意见** | 🟡 警告 | 仔细阅读保留事项 |
| **否定意见** | 🔴 高危 | 一票否决，不投资 |
| **无法表示意见** | 🔴 高危 | 一票否决，不投资 |
| **带强调事项段** | 🟡 注意 | 关注强调内容 |

**其他信号**：
- 频繁更换会计师事务所 ⚠️
- 审计费用异常下降 ⚠️
- 审计师离职率过高 ⚠️

### 3.2 财务数据异常信号

**利润表异常**：
- [ ] 毛利率显著高于同行，且无合理解释
- [ ] 营收增速远超行业，但市占率未提升
- [ ] 费用率突然大幅下降
- [ ] 非经常性损益占比>30%
- [ ] 扣非净利润与净利润差异巨大

**资产负债表异常**：
- [ ] 应收账款增速 > 营收增速
- [ ] 存货周转率持续下降但营收增长
- [ ] 在建工程长期挂账（>2年不转固）
- [ ] 其他应收款/其他应付款金额异常大
- [ ] 商誉/无形资产占比>30%
- [ ] 存贷双高（货币资金和有息负债都高）

**现金流量表异常**：
- [ ] 经营现金流长期为负或远低于净利润
- [ ] 收现比持续<0.9
- [ ] 筹资活动现金流异常活跃（频繁融资）
- [ ] 投资现金流与主业无关的大额往来

### 3.3 关联交易与治理异常

- [ ] 前五大客户/供应商集中度>80%
- [ ] 关联交易金额大且定价不公允
- [ ] 实控人高比例质押（>70%）
- [ ] 高管频繁减持套现
- [ ] 财务总监频繁更换
- [ ] 独立董事集体辞职
- [ ] 被证监会立案调查或处罚

### 3.4 商业模式异常

- [ ] 客户多为新成立公司或自然人
- [ ] 海外业务占比高且无法核实
- [ ] 商业模式复杂难懂
- [ ] 频繁跨行业并购
- [ ] 主业亏损但靠非主业盈利

## 五、行业调整系数

不同行业的财务指标标准不同，评分时需考虑行业特性：

| 行业类型 | 特点 | 调整说明 |
|---------|------|---------|
| **白酒/医药** | 高毛利、低周转 | 毛利率权重↑，周转率权重↓ |
| **零售/贸易** | 低毛利、高周转 | 周转率权重↑，毛利率权重↓ |
| **银行/保险** | 高杠杆 | 杠杆因素调整，关注ROA |
| **制造业** | 重资产 | 关注折旧、资本开支 |
| **科技/软件** | 轻资产、高研发 | 关注研发费用、无形资产 |
| **房地产** | 预售制、高杠杆 | 关注预收款、土储 |
| **公用事业** | 稳定现金流 | 关注分红率、ROE稳定性 |

## 八、常见误区提醒

1. **只看利润不看现金流** → 可能踩雷
2. **只看绝对值不看趋势** → 可能买在顶点
3. **只看个体不看行业** → 可能选中夕阳行业龙头
4. **只看财务不看商业模式** → 可能看不懂护城河
5. **只看历史不看未来** → 可能错过转折点
