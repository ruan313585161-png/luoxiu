## 🔄 使用流程建议

### 完整分析流程（推荐）

```
第一步：基础信息
├── 用户画像（了解风险偏好）
└── 持仓记录（了解现有仓位）

第二步：财报分析
└── 财报评分框架（五维评估）

第三步：大师框架（可单选或多选）
├── 巴菲特框架（判断质量）
├── 彼得·林奇框架（判断成长性）
├── 霍华德·马克斯框架（判断周期）
├── 瑞·达利欧框架（宏观视角）
└── 芒格框架（检查清单+心理偏见）← 必做

第四步：技术分析
└── 确定买卖点

第五步：产业链
└── 行业比较和定位
```

### 快速分析流程

```
财报评分 → 芒格检查清单 → 技术分析
（15分钟）  （10分钟）      （5分钟）
```

## 💡 芒格框架的独特价值

**为什么芒格框架是必做的？**

芒格框架不同于其他四位大师：

1. **不是用来发现机会，而是用来避免错误**
2. **强调心理偏见控制**（25条人类误判心理学）
3. **提供检查清单**（确保不遗漏关键要素）
4. **逆向思维**（先想怎么失败）

**建议：无论使用哪个大师框架，最后都要过一遍芒格检查清单。**
