## 一、技术分析的三大支柱

### 1.1 市场行为包容消化一切
- **价格反映所有信息**：基本面、消息面、情绪面都已反映在价格中
- **不预测，只跟随**：识别趋势并顺势而为
- **历史会重演**：市场心理和行为模式重复出现

### 1.2 三大分析维度

| 维度 | 工具 | 适用场景 |
|------|------|---------|
| **形态分析** | K线形态、图表形态 | 判断反转/持续 |
| **趋势分析** | 均线、趋势线、通道 | 确定方向和强度 |
| **量价分析** | 成交量、持仓量 | 确认信号有效性 |

## 二、日本蜡烛图技术（史蒂夫·尼森）

### 2.1 K线基础构成

```
        上影线（最高价-实体顶部）
           │
        ┌──┴──┐ ← 实体（开盘价-收盘价）
        │     │
        └──┬──┘
           │
        下影线（实体底部-最低价）
```

**实体含义**：
- **阳线（红色）**：收盘价 > 开盘价，买方占优
- **阴线（绿色）**：收盘价 < 开盘价，卖方占优
- **实体越大**：力量越强
- **实体越小**：多空平衡（十字星）

### 2.2 单根K线形态

| 形态 | 特征 | 含义 |
|------|------|------|
| **锤子线** | 小实体，长下影线，无/短上影线 | 底部反转信号 |
| **吊人线** | 形态同锤子线，出现在顶部 | 顶部反转信号 |
| **吞没形态** | 后一根完全包住前一根 | 强势反转信号 |
| **十字星** | 开盘价=收盘价，无实体 | 多空平衡，警惕变盘 |
| **长阳线** | 大阳线，短影线 | 强烈看涨 |
| **长阴线** | 大阴线，短影线 | 强烈看跌 |

### 2.3 多根K线组合

#### 反转形态

| 形态 | 构成 | 确认条件 | 目标价位 |
|------|------|---------|---------|
| **早晨之星** | 长阴+十字星+长阳 | 第三根突破第一根中点 | 第一根高点 |
| **黄昏之星** | 长阳+十字星+长阴 | 第三根跌破第一根中点 | 第一根低点 |
| **刺透形态** | 长阴+刺入前一根50%+的阳线 | 阳线实体过半 | 前高 |
| **乌云盖顶** | 长阳+深入50%+的阴线 | 阴线实体过半 | 前低 |
| **启明星** | 下跌+十字星+上涨 | 第三根突破第一根实体 | 前期高点 |

#### 持续形态

| 形态 | 特征 | 操作 |
|------|------|------|
| **上升三法** | 长阳+3小阴+长阳突破 | 做多 |
| **下降三法** | 长阴+3小阳+长阴突破 | 做空 |
| **旗形整理** | 快速运动后的小幅整理 | 顺势加仓 |
| **三角整理** | 波动收窄 | 等待突破 |

### 2.4 蜡烛图实战 checklist

**买入信号 checklist** ✅
- [ ] 出现锤子线/早晨之星/刺透形态
- [ ] 形态出现在支撑位（均线/前期低点）
- [ ] 成交量放大确认
- [ ] 下一根K线继续收阳确认

**卖出信号 checklist** ⚠️
- [ ] 出现吊人线/黄昏之星/乌云盖顶
- [ ] 形态出现在阻力位（均线/前期高点）
- [ ] 成交量放大确认
- [ ] 下一根K线继续收阴确认

## 三、趋势分析（约翰·墨菲）

### 3.1 趋势的定义与分类

```
上升趋势：高点越来越高，低点越来越高
    ↗️ 高点
   ↗️ 低点
  ↗️ 高点
 ↗️ 低点
↗️

下降趋势：低点越来越低，高点越来越低
↘️
 ↘️ 高点
  ↘️ 低点
   ↘️ 高点
    ↘️ 低点

横盘趋势：高点和低点基本水平
───────
       ───────
───────
```

### 3.2 趋势线与通道

**画法规则**：
- **上升趋势线**：连接两个以上低点
- **下降趋势线**：连接两个以上高点
- **通道线**：与趋势线平行的另一条线

**有效性判断**：
- 触点越多，趋势线越有效
- 角度45度最佳，过陡易破，过缓滞后
- 突破趋势线后，原线变支撑/阻力

### 3.3 支撑与阻力

| 概念 | 定义 | 转换规则 |
|------|------|---------|
| **支撑位** | 价格下跌时遇到的买盘区域 | 跌破后变阻力 |
| **阻力位** | 价格上涨时遇到的卖盘区域 | 突破后变支撑 |

**关键支撑/阻力来源**：
1. 前期高低点
2. 整数关口（如3000点）
3. 均线（MA20/MA60/MA120）
4. 趋势线
5. 黄金分割位

### 3.4 移动平均线系统

| 均线 | 周期 | 作用 |
|------|------|------|
| MA5 | 5日 | 短期趋势，日内交易参考 |
| MA10 | 10日 | 短期趋势，短线操作参考 |
| MA20 | 20日 | 中期趋势，波段操作参考 |
| MA60 | 60日 | 中期趋势，季线 |
| MA120 | 120日 | 长期趋势，半年线 |
| MA250 | 250日 | 长期趋势，年线 |

**均线买卖法则**（葛兰碧八大法则）：
1. 均线从下降转平，价格上穿均线 → 买
2. 价格在均线上方，回调不破均线 → 买
3. 价格跌破均线，但均线仍向上 → 买
4. 价格暴跌远离均线 → 买（反弹）
5. 均线从上升转平，价格下穿均线 → 卖
6. 价格在均线下方，反弹不破均线 → 卖
7. 价格突破均线，但均线仍向下 → 卖
8. 价格暴涨远离均线 → 卖（回调）

### 3.5 技术指标

#### MACD（指数平滑异同平均线）

```
构成：
- DIF（快线）= 12日EMA - 26日EMA
- DEA（慢线）= DIF的9日EMA
- MACD柱 = 2×(DIF-DEA)

用法：
- 金叉（DIF上穿DEA）：买入信号
- 死叉（DIF下穿DEA）：卖出信号
- 顶背离：价格新高，MACD未新高 → 警惕见顶
- 底背离：价格新低，MACD未新低 → 警惕见底
```

#### RSI（相对强弱指数）

```
公式：RSI = 100 - (100 / (1 + RS))
RS = n日内上涨平均幅度 / n日内下跌平均幅度

用法：
- RSI > 70：超买，警惕回调
- RSI < 30：超卖，警惕反弹
- 顶背离：价格新高，RSI未新高
- 底背离：价格新低，RSI未新低
```

#### 布林带（BOLL）

```
构成：
- 中轨 = 20日均线
- 上轨 = 中轨 + 2倍标准差
- 下轨 = 中轨 - 2倍标准差

用法：
- 价格触及上轨：超买，可能回调
- 价格触及下轨：超卖，可能反弹
- 缩口：波动减小，等待方向选择
- 开口：波动加大，趋势可能加速
```

## 四、缠论（缠中说禅）

### 4.1 缠论核心概念

**基本元素**：
- **分型**：顶分型（高点）、底分型（低点）
- **笔**：连接相邻顶底分型的线段
- **线段**：至少三笔构成的走势段
- **中枢**：三个连续次级别走势重叠部分

### 4.2 走势类型

| 类型 | 定义 | 操作 |
|------|------|------|
| **上涨** | 高点越来越高，低点越来越高 | 持股或逢低买入 |
| **下跌** | 低点越来越低，高点越来越低 | 空仓或逢高卖出 |
| **盘整** | 中枢震荡，无明显趋势 | 高抛低吸或观望 |

### 4.3 三类买卖点

```
一买：下跌背驰后的第一个买点（趋势转折）
       ↓
二买：一买后回调不破低的买点（确认趋势）
       ↓
三买：突破中枢后回抽不破中枢的买点（加速上涨）
```

| 买点 | 位置 | 风险 | 收益 |
|------|------|------|------|
| **第一类买点** | 趋势底部 | 可能继续下跌 | 最高 |
| **第二类买点** | 确认底部 | 中等 | 中高 |
| **第三类买点** | 趋势加速 | 较低 | 中等 |

### 4.4 背驰判断

**定义**：价格创新高/新低，但力度指标未创新高/新低

**判断方法**：
1. **MACD背驰**：价格新高，MACD红柱面积/高度未新高
2. **均线背驰**：价格新高，均线斜率减缓
3. **成交量背驰**：价格新高，成交量萎缩

**操作**：
- 顶背驰 → 卖出
- 底背驰 → 买入

### 4.5 缠论实战 checklist

**寻找一买 checklist**：
- [ ] 已完成下跌趋势
- [ ] 出现底分型
- [ ] 背驰确认（MACD/均线）
- [ ] 次级别出现买点
- [ ] 设置止损：跌破前低

**寻找三买 checklist**：
- [ ] 已突破中枢上轨
- [ ] 回抽不破中枢
- [ ] 次级别出现买点
- [ ] 成交量配合放大

## 五、综合分析框架

### 5.1 多时间周期分析

| 周期 | 作用 | 适用人群 |
|------|------|---------|
| **月线** | 判断大方向 | 长期投资者 |
| **周线** | 判断中期趋势 | 波段投资者 |
| **日线** | 主要操作周期 | 普通投资者 |
| **60分钟** | 精确买卖点 | 短线投资者 |
| **15分钟** | 日内交易 | 超短线投资者 |

**原则**：
- 大周期定方向，小周期找买点
- 大周期趋势向上，小周期回调买入
- 大周期趋势向下，小周期反弹卖出

### 5.2 综合分析流程

```
第一步：判断大势（日线/周线）
    ↓ 趋势向上/向下/盘整
第二步：寻找支撑/阻力位
    ↓ 均线、前期高低点、趋势线
第三步：观察K线形态
    ↓ 反转形态/持续形态
第四步：技术指标确认
    ↓ MACD、RSI、布林带
第五步：成交量验证
    ↓ 放量确认，缩量观望
第六步：制定交易计划
    ↓ 入场点、止损位、目标位
```

## 六、风险控制

### 6.1 止损原则

| 止损类型 | 方法 | 适用场景 |
|---------|------|---------|
| **固定比例** | 亏损达5-8%止损 | 所有交易 |
| **技术止损** | 跌破支撑位/均线止损 | 技术交易 |
| **时间止损** | 持仓3-5天不达预期止损 | 短线交易 |
| **移动止损** | 盈利后上移止损位 | 趋势跟踪 |

### 6.2 仓位管理

- **单笔交易**：不超过总资金10%
- **单只股票**：不超过总资金20%
- **总仓位**：牛市80%，震荡市50%，熊市20%

### 6.3 技术分析的局限性

⚠️ **注意**：
1. 技术分析是概率游戏，不是确定性预测
2. 突发消息可能破坏技术形态
3. 震荡市技术指标容易失效
4. 需要结合基本面和资金管理

## 九、实战 checklist

**每日复盘 checklist**：
- [ ] 检查持仓股票技术形态变化
- [ ] 更新支撑阻力位
- [ ] 观察大盘趋势
- [ ] 复盘当日交易
- [ ] 制定明日计划

**买入前 checklist**：
- [ ] 大势是否支持
- [ ] 是否处于支撑位
- [ ] K线形态是否支持
- [ ] 指标是否支持
- [ ] 止损位是否明确
- [ ] 盈亏比是否合理（>2:1）

**卖出前 checklist**：
- [ ] 是否触及止损
- [ ] 是否到达目标位
- [ ] 是否出现反转信号
- [ ] 趋势是否改变
- [ ] 是否有更好的机会
