### 详细记录：文件发送功能（2026-03-13 18:15）

#### 背景
昨天（2026-03-12）测试多次尝试发送文件到邮件均未成功。

#### 昨天失败原因分析
| 失败方式 | 原因 |
|----------|------|
| ❌ 用 `exec` + `curl` 绕路 | 绕过官方通道，参数复杂易错 |
| ❌ 未使用 `message` 工具原生功能 | 不知道 `message` 支持 `filePath` 参数 |
| ❌ 可能路径或权限问题 | 未确认文件绝对路径和可读性 |

#### 今天成功方法

**方法 1：聊天窗口发送（18:15）**  
**工具**: `message`  
**Action**: `send`  
**关键参数**:
```json
{
  "action": "send",
  "filePath": "/绝对路径/到/文件.md",
  "filename": "显示给用户的文件名.md",
  "mimeType": "text/markdown",
  "message": "邮件正文说明"
}
```
**结果**: ✅ 成功送达聊天窗口

**方法 2：邮件发送（18:45）**  
**工具**: Python `smtplib`  
**关键配置**:
- SMTP服务器: `smtp.qq.com:465`
- 授权码: **存储在 `.env` 文件**
- 必须使用 SSL: `SMTP_SSL`

**结果**: ✅ 成功发送到 313585161@qq.com

#### 核心要点
- ✅ 聊天窗口：直接用 `message` 工具的官方 `filePath` 参数
- ✅ 邮件发送：Python `smtplib` + 正确授权码 + SSL
- ✅ 使用**绝对路径**（如 `/root/.openclaw/workspace/...`）
- ✅ 明确指定 `mimeType`（markdown用 `text/markdown`）
