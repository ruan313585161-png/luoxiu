## 一、雪球方丈（方三文）——资产配置派

### 1.1 人物简介
- **身份**：雪球创始人、雷石价值基金基金经理
- **背景**：北大中文系出身，曾任《南方周末》记者
- **代表产品**：雷石价值基金

### 1.2 核心理念：跨市场资产配置

**"三分法"资产配置框架**：
```
偏债类资产    35%
偏股类资产    60%
商品类资产     5%
```

**核心理念**：
- **"投资这个星球上的主要资产，你就是大盘本身"**
- 全球化配置：A股、港股、美股跨市场分散
- 跨资产配置：股票、债券、商品组合

### 1.3 好公司的三大标准

| 标准 | 解释 | 典型公司 |
|------|------|---------|
| **利润规模大** | 显著盈利能力和行业地位 | 腾讯、阿里 |
| **盈利持续性** | 商业模式支撑长期赚钱 | 茅台、Meta |
| **盈利难度低** | 有护城河，无需高投入维持 | 高端白酒、互联网平台 |

### 1.4 操作原则

1. **不调仓策略**
   - "大几年不调仓"
   - 降低摩擦成本，避免情绪干扰
   - 持有腾讯、Meta等超十年

2. **定投再平衡**
   - 用持续收入定期投资
   - 资产比例偏离时恢复原始配置
   - 自动实现"量化高抛低吸"

3. **估值态度**
   - 99%的时候说"合理"
   - 极端机会罕见，不抱抄底幻想
   - "市场绝大多数时候估值都是合理的"

### 1.5 方丈 checklist

**选股 checklist**：
- [ ] 是否为行业龙头？
- [ ] 商业模式是否简单可理解？
- [ ] 是否有护城河（品牌/规模效应）？
- [ ] 管理层是否低调务实？

**心态 checklist**：
- [ ] 是否接受自己是普通人？
- [ ] 是否放弃战胜市场的执念？
- [ ] 是否能忍受长期持有？
- [ ] 是否降低收益预期？

## 二、张坤（易方达）——高质量价值派

### 2.1 人物简介
- **身份**：易方达基金副总经理（已卸任专注投资）、基金经理
- **背景**：清华大学研究生毕业，2008年加入易方达
- **管理规模**：约600亿（国内最大主动权益基金经理）
- **代表产品**：易方达蓝筹精选、易方达中小盘

### 2.2 投资框架

**核心理念**：
> "企业的价值是其生命周期内所有自由现金流的折现"

**选股三要素**：
1. **商业模式优秀**
2. **竞争力显著**
3. **定价能力强**

### 2.3 投资风格特征

| 特征 | 具体表现 | 案例 |
|------|---------|------|
| **高仓位运作** | 股票仓位90%以上 | 不择时，用选股对冲风险 |
| **高集中度** | 前十大重仓占80%+ | 重仓茅台、腾讯等 |
| **低换手率** | 年换手率30-50% | 持有茅台24个季度 |
| **长期持有** | 持股周期3-5年起 | 五粮液持有超5年 |

### 2.4 能力圈与持仓

**核心能力圈**：
- 消费（食品饮料）
- 医药
- 互联网科技
- 金融

**经典重仓**：
- 贵州茅台、五粮液、泸州老窖（白酒）
- 腾讯控股、阿里巴巴（互联网）
- 招商银行、平安银行（金融）

### 2.5 张坤 checklist

**选股 checklist**：
- [ ] 是否能产生自由现金流？
- [ ] 护城河是否持续加深？
- [ ] 管理层是否对股东友好？
- [ ] 行业空间是否广阔？

**持有 checklist**：
- [ ] 长期持有，忽略短期波动
- [ ] 不因高估而卖出优秀公司
- [ ] 陪伴企业成长5-10年

## 三、朱少醒（富国基金）——成长精选派

### 3.1 人物简介
- **身份**：富国基金研究部总经理、基金经理
- **管理年限**：19年（2005年至今）
- **代表产品**：富国天惠成长混合
- **历史业绩**：19年17倍，年化约19%

### 3.2 投资理念

**一句话总结**：
> "淡化择时、高仓位运转、均衡配置、精研个股"

**自下而上精选成长股**：
- 偏好具有良好"企业基因"的公司
- 公司治理结构完善
- 管理层优秀
- 分享企业自身增长收益

### 3.3 投资风格特征

| 特征 | 具体表现 | 说明 |
|------|---------|------|
| **淡化择时** | 仓位基本90%以上 | 2010年后放弃择时 |
| **均衡配置** | 单一行业不超过20% | 食品饮料、医药、科技等 |
| **精选个股** | 自下而上选股 | 先看公司，再看行业 |
| **长期持有** | 国瓷材料持有6年涨11倍 | 陪伴企业成长 |
| **高换手率** | 相对张坤略高 | 季度调仓优化 |

### 3.4 行业配置

**偏好行业**：
- 食品饮料（白酒）
- 医药生物
- 计算机（软件）
- 商贸零售（早期）
- 银行、房地产

**经典案例**：
- **贵州茅台**：2006-2012连续7年前十大重仓
- **国瓷材料**：2014-2020持有6年，涨11倍

### 3.5 朱少醒 checklist

**选股 checklist**：
- [ ] 是否有良好"企业基因"？
- [ ] 公司治理是否完善？
- [ ] 管理层是否优秀？
- [ ] 能否分享经济增长成果？

**风险控制**：
- [ ] 单一行业不超过20%
- [ ] 高仓位但不集中持股
- [ ] 分散投资平滑波动

## 四、谢治宇（兴证全球基金）——性价比平衡派

### 4.1 人物简介
- **身份**：兴证全球基金投资总监、基金经理
- **背景**：复旦大学经济学硕士，2007年加入兴全
- **管理规模**：约700亿
- **代表产品**：兴全合润、兴全合宜
- **历史业绩**：9年540%+，年化22.55%

### 4.2 核心投资框架

**一句话总结**：
> "寻找最具性价比的个股"

**框架演进**：
```
阶段一：性价比优先（2013-2016）
    ↓ 问题：好公司持有时间不够长
阶段二：好公司优先（2017-2019）
    ↓ 问题：忽视估值风险
阶段三：好公司+性价比（2020至今）
    当前框架：好公司优先，但必须有性价比
```

### 4.3 投资风格特征

| 特征 | 具体表现 | 说明 |
|------|---------|------|
| **均衡配置** | 前十大重仓30-50% | 不押注单一行业 |
| **自下而上** | 精选个股，不看赛道 | 周期股也能买出成长性 |
| **成长偏好** | 注重增速与估值匹配 | 不贴价值/成长标签 |
| **高仓位** | 常年90%以上 | 不择时 |
| **风险控制** | 近5年最大回撤-22% | 好于同类平均 |

### 4.4 选股逻辑

**从增速与估值匹配度出发**：
- 不拘泥于板块（周期/消费/科技）
- 注重"性价比"
- 加大"确定性"权重
- 没有业绩支撑的标的果断替换

**持仓特点**：
- 行业分散（前十大重仓常覆盖8个行业）
- 个股多元（医药、家电、银行、电子等）
- 持股周期较长（平均4.2个季度）

### 4.5 谢治宇 checklist

**选股 checklist**：
- [ ] 是否是好公司？
- [ ] 增速与估值是否匹配？
- [ ] 确定性是否足够？
- [ ] 性价比是否合理？

**组合构建 checklist**：
- [ ] 行业是否均衡？
- [ ] 个股是否分散？
- [ ] 风险是否可控？
- [ ] 长期价值是否明确？

## 五、董承非（睿郡资产）——风险收益匹配派

### 4.1 人物简介
- **身份**：睿郡资产管理合伙人（原兴全基金副总经理）
- **管理年限**：15年（2007-2022公募，2022至今私募）
- **代表产品**：兴全趋势投资、兴全新视野
- **历史业绩**：15年年化约19%，为基民赚取超380亿

### 5.2 核心理念：策略型选手

**一句话总结**：
> "找到最佳的风险收益匹配，而不只是局限在选股"

**与纯选股者的区别**：
- 纯选股者：信仰公司，穿越牛熊
- 策略型选手：用策略回避风险或赚钱

### 5.3 投资框架

**三维比较体系**：
```
第一层：大类资产比较
    股票 vs 债券 vs 现金 vs 可转债
    
第二层：行业比较
    不同行业的性价比
    
第三层：个股比较
    同行业内的性价比
```

### 5.4 投资风格特征

| 特征 | 具体表现 | 案例 |
|------|---------|------|
| **会做择时** | 极端行情减仓 | 2015年仓位降至32% |
| **均衡分散** | 行业不集中 | 食品饮料10%、银行9%等 |
| **重视风控** | 先想风险再想收益 | 2008年仅跌44%（沪深300跌65%）|
| **大类资产配置** | 可转债、套利、定增 | 灵活使用多种工具 |
| **长期持有** | 恒立液压持有多年 | 挖掘行业龙头 |

**历史择时案例**：
- **2008年**：股票仓位降至50%附近
- **2015年**：三季度仓位降至32%
- **2021年初**：提出警惕牛熊转换风险

### 5.5 董承非 checklist

**建仓 checklist**：
- [ ] 市场是否可为？
- [ ] 投入多少比例能承受？
- [ ] 风险收益是否匹配？
- [ ] 万一错了怎么办？

**风控 checklist**：
- [ ] 是否考虑了所有可能风险？
- [ ] 是否有安全边际？
- [ ] 极端情况下能否承受？
- [ ] 是否追求中庸而非极端？
