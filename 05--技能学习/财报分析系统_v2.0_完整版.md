## 一、核心标准（巴菲特原则）

### 1.1 ROE与ROIC标准

| 等级 | ROE标准 | ROIC标准 | 说明 |
|------|---------|----------|------|
| 卓越 | ≥30% | ≥15%（5年）| 凤毛麟角 |
| 优秀 | ≥20%（5年）| ≥12%（3年）| 卓越公司 |
| 良好 | ≥15%（3年）| - | 优秀公司 |
| 行业调整 | 银行≥12%,白酒≥20% | - | 去杠杆后真实盈利 |

### 1.2 现金流铁律

| 指标 | 优秀标准 | 危险信号 |
|------|---------|---------|
| 经营现金流/净利润 | ≥1.0 | <0.8 |
| 自由现金流/净利润 | >0.8 | <0.5 |
| 现金分红率 | 连续3年≥30% | 无分红 |
| 分红安全垫 | ≤经营现金流50% | >经营现金流 |

### 1.3 盈利能力指标

| 指标 | 卓越 | 优秀 | 良好 | 一般 | 危险 |
|------|------|------|------|------|------|
| 毛利率 | >50% | >40% | 25-40% | 15-25% | <15% |
| 净利率 | >25% | >20% | 10-20% | 5-10% | <5% |
| ROE | >25% | >20% | 15-20% | 10-15% | <10% |

## 二、五维财报分析框架

### 2.1 盈利能力（25%）

**核心指标**：
- ROE、ROIC、毛利率、净利率
- **研发投入占比**（科技行业>10%）
- **资本化率**（科技行业<30%）

### 2.2 偿债能力（20%）

| 指标 | 优秀 | 安全 | 危险 |
|------|------|------|------|
| 资产负债率 | <40% | 40-60% | >70% |
| 有息负债率 | <20% | 20-40% | >50% |
| 流动比率 | >2 | 1.5-2 | <1 |

**v2.0新增**：
- 短期借款/流动负债 <30%
- 受限货币资金/货币资金 <10%

### 2.3 营运能力（15%）

- 应收账款周转率、存货周转率
- **应付账款周转率**（制造业>6次）
- **产成品/存货 <30%**
- 1年以内应收账款占比 >90%

### 2.4 成长能力（20%）

- 营收增长率、净利润增长率
- **内生增长占比 >80%**
- 市占率年提升 ≥2%
- 3年波动率 <15%

### 2.5 现金流健康度（20%）

- 经营现金流/净利润
- 自由现金流
- **3年经营现金流波动率 <20%**
- 现金收益率 >1.5%

**现金流组合判断**：
```
+ - - = 奶牛型（最佳）⭐
+ - + = 成长期（扩张）✅
+ + - = 成熟期（健康）✅
+ + + = 抽血型（危险）🚨
```

## 三、财务造假排雷清单

### 3.1 审计意见异常

| 意见类型 | 风险等级 |
|---------|---------|
| 标准无保留 | 🟢 安全 |
| 保留意见 | 🟡 警告 |
| 否定/无法表示意见 | 🔴 高危（一票否决）|

**v2.0新增**：
- 审计师频繁更换（<3年）→ 黄色警告
- 审计费用变动 >50% → 警告

### 3.2 财务数据异常（量化阈值）

| 异常信号 | 阈值 | 风险等级 |
|---------|------|---------|
| 毛利率单季度变动 | >20%且无解释 | 🔴 高危 |
| 在建工程/总资产 | >20%且2年不转固 | 🔴 高危 |
| 商誉减值/净利润 | >50% | 🔴 高危 |
| 经营现金流同比变动 | >100%且与营收背离 | 🟡 警告 |

### 3.3 关联交易与治理异常

| 指标 | v2.0阈值 | 风险等级 |
|------|---------|---------|
| 单一客户占比 | >50% | 🔴 高危 |
| 实控人质押（流通股）| >50% | 🔴 高危 |
| 高管减持/年薪 | >10倍 | 🟡 警告 |

### 3.4 财务口径异常（v2.0新增）

- 会计准则变更后指标大幅波动
- 合并范围一年内变动>3家且影响净利润>20%
- 母公司vs合并报表资金差异>50%

## 四、财报评分体系（百分制）

### 4.1 维度权重

| 维度 | 权重 | 满分 |
|------|------|------|
| 盈利能力 | 25% | 25分 |
| 偿债能力 | 20% | 20分 |
| 营运能力 | 15% | 15分 |
| 成长能力 | 20% | 20分 |
| 现金流健康度 | 20% | 20分 |
| **附加分** | - | +5分 |

### 4.2 趋势调整分（±10%）

- 指标持续改善 → +10%
- 指标持续恶化 → -10%
- 指标大幅波动 → -5%

### 4.3 附加分（0-5分）

| 指标 | 5分 | 3分 | 0分 |
|------|-----|-----|-----|
| ESG评级 | A/A- | B+/B | B-及以下 |
| 分红指标 | 连续5年≥30% | 连续3年≥30% | 其他 |
| 研发（科技）| >15%且费用化 | 匹配行业均值 | 其他 |

### 4.4 总分评级

| 总分 | 评级 | 建议 |
|------|------|------|
| 90-105分 | ⭐⭐⭐⭐⭐ 卓越 | 重点关注 |
| 80-89分 | ⭐⭐⭐⭐ 优秀 | 值得投资 |
| 70-79分 | ⭐⭐⭐ 良好 | 适度配置 |
| 60-69分 | ⭐⭐ 一般 | 谨慎投资 |
| <60分 | ❌ 危险 | 远离 |

## 六、细分赛道极致适配（backlog）

| 赛道 | 新增核心指标 | 权重调整 |
|------|-------------|---------|
| **生物医药（临床阶段）** | 研发管线进度、资本化率、license-in/out | 研发投入+15%，不考核净利润 |
| **周期股（煤炭/有色）** | PB-ROE分位、产能利用率、库存周转 | 成长能力-10%，营运能力+25% |
| **跨境电商** | 汇率对冲比例、海外坏账率、汇兑损益 | 现金流波动率放宽至30% |
| **金融科技** | ARPU、不良率、监管评级 | 剔除资产负债率，新增合规成本/营收 |

## 七、企业生命周期权重（backlog）

| 阶段 | 权重调整方向 | 核心逻辑 |
|------|-------------|---------|
| **初创期（未盈利）** | 研发+10%、现金流+5%，不考核ROE/分红 | 优先看"活下去"的能力 |
| **成长期** | 内生增长+5%、客户集中度-5% | 关注增长质量，规避大客户依赖 |
| **成熟期** | 分红安全垫+5%、现金收益率+5% | 关注"现金奶牛"属性 |
| **衰退期** | 有息负债率+5%、资产处置收益扣分 | 警惕"卖资产保利润"的伪盈利 |

## 八、案例库

### 8.1 正面案例

**宁德时代（科技/新能源）**
- ROE 15%+、研发投入>10%、经营现金流/净利润>1.2、产能利用率>80%

**恒瑞医药（医药）**
- 研发投入>15%、毛利率>85%、ROE 15%+、经营现金流稳定

### 8.2 负面案例

**康美药业（财务造假典型）**
- 存贷双高（货币资金300亿+有息负债200亿）
- 经营现金流与净利润背离（净利润40亿，现金流-30亿）
- 审计意见：无法表示意见

**辅仁药业（资金链断裂）**
- 账面18亿货币资金无法动用
- 连续3年经营现金流为负
- 分红违约暴露资金链断裂

## 九、误区提醒（12条完整版）

| 误区 | 表现 | 应对 |
|------|------|------|
| 1. 只看净利润不看扣非 | 利润靠补贴 | 看扣非归母净利润 |
| 2. 只看利润不看现金流 | 纸面盈利 | 经营现金流/净利润>1 |
| 3. 只看ROE高低不看质量 | 高杠杆ROE | 拆解ROE三因素 |
| 4. 只看绝对值不看行业对比 | 跨行业误判 | 同业对比看排名 |
| 5. 只看短期不看长期 | 被噪音干扰 | 看3年趋势 |
| 6. 只看合并报表忽略母公司 | 资金被占用 | 母公司vs合并差异>50%高危 |
| 7. 忽略财务指标口径变化 | 会计政策变更 | 关注重要会计政策变更 |
| 8. 过度关注单季度数据 | 季节性误导 | 看同比+年度数据 |
| 9. 把"高增长"等同于"高价值" | 伪成长 | 看现金流+内生增长 |
| 10. 忽略分部报告（backlog）| 核心分部亏损 | 分部亏损>10%整体-5% |
| 11. 混淆会计利润与经济利润（backlog）| ROIC<WACC | 经济利润<0时-10% |
| 12. 忽视隐性负债（backlog）| 未计提租赁负债 | IFRS16还原租赁负债 |

## 十、Excel公式模板

### 核心指标公式

```excel
# ROE
=B2/AVERAGE(B3:C3)*100

# 毛利率
=(B4-B5)/B4*100

# 经营现金流/净利润
=B6/B2

# 资产负债率
=B8/AVERAGE(B7:C7)*100

# 应收账款周转率
=B4/AVERAGE(B11:C11)
```

### 评分自动化

```excel
# ROE得分（25分满分）
=IF(D2>=20,5,IF(D2>=15,4,IF(D2>=10,3,IF(D2>=5,2,1))))*5

# 趋势调整
=IF(AND(本期>上期,上期>上上期),1.1,IF(AND(本期<上期,上期<上上期),0.9,1))
```
