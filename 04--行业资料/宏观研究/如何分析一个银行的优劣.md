1 首先要看会几个指标,第一个指标

核充率是核心一级资本充足率的简称，是银行监管中最核心、最严格的资本指标。

                    │   总资本充足率   │  ≥ 10.5%（非系统重要性银行）

                    │  (Capital Ratio) │  ≥ 11.5%-13.5%（系统重要性银行）

              ┌──────────────┼──────────────┐

              │              │              │

        二级资本      一级资本充足率      其他一级资本

       (Tier 2)     (Tier 1 Ratio)      (AT1/优先股)

       ≥ 2%           ≥ 8.5%              ≥ 1%

                             │

                    ┌────────┴────────┐

                    │  核心一级资本充足率 │  ≥ 7.5%（最低标准）

                    └─────────────────┘

1. 核心一级资本充足率（核充率/CET1）

定义：银行最核心、质量最高的资本（普通股、留存收益、一般风险准备）÷ 风险加权资产（RWA）

监管红线：≥ 7.5%（最低要求）

实际安全线：≥ 9.0%（考虑储备资本和逆周期缓冲后）

为什么最重要？

只能由普通股和利润留存构成

不能包含优先股、永续债、二级资本债等

是银行吸收损失的第一道防线

2. 一级资本充足率（Tier 1）

构成：核心一级资本 + 其他一级资本（优先股、永续债）

监管要求：≥ 8.5%

作用：包含混合资本工具，质量次于核充

3. 总资本充足率（CAR）

构成：一级资本 + 二级资本（次级债、超额贷款损失准备）

监管要求：≥ 10.5%

作用：包含可以减记或转股的债务工具

第三组 (3家)： 交通银行、招商银行、兴业银行。

第四组 (4家)： 中国工商银行、中国银行、中国建设银行、中国农业银行。

投资博弈提示

安全边际： 银行实际的核心一级资本充足率通常会高出最低要求 1-2 个百分点，作为缓冲空间。

分红影响： 如果一家银行的核心一级资本充足率逼近上述红线（例如江苏银行目前在 8.5% 左右，非常接近 7.75% 的红线），它可能会通过限制分红或进行再融资（如发行可转债）来补充资本，这正是您之前关注的“核充焦虑”的来源

虽然监管最低要求是7.0%-8.5%，但**9.5%是实际操作中的安全底线**

但银行需要预留：
RWA增长缓冲：如果银行资产增速10%，需要额外0.5-1%核充
监管处罚缓冲：低于10%会被限制分红、限制扩张
市场信心缓冲：低于9%会被视为高风险，融资成本上升
因此：
>11%：优秀，可以高分红+高扩张
9.5%-11%：正常，可以维持经营
<9.5%：紧张，可能面临限制分红或被迫融资
<8.5%：危险，监管可能介入要求整改
核充率（核心一级资本充足率）是银行真正的"家底"，
核充率=普通股+资本公积+盈余公积+未分配利润+一般风险准备/风险加权资产/100%

1. 普通股（实收资本/股本）

通俗说法：股东真金白银投进来的本金
具体构成：
1. IPO时股民买股票的钱（1元面值部分）
2. 增发、配股时新股东投的钱
3. 可转债转股形成的股本
例如:宁波银行总股本66亿股，每股面值1元，这部分就是66亿元。这是股东最原始的投资，亏光了就没了
2. 资本公积（股本溢价）
通俗说法：股票卖贵了的部分
具体构成：
IPO发行价6元，面值1元，多出的5元进资本公积
增发价20元，面值1元，多出的19元进资本公积
例子：
如果宁波银行增发1亿股，发行价25元，那么：
股本增加：1亿元（面值）
资本公积增加：24亿元（溢价）
这24亿也是核心一级资本，但不能分红，只能留着抵御风险
3. 盈余公积（法定+任意公积金）
通俗说法：强制存钱罐（法律规定必须留的钱）
提取规则：
法定盈余公积：每年净利润的10%必须提取，直到累计达到注册资本的50%
任意盈余公积：股东会决定额外存的钱
用途：弥补亏损  转增股本（送股）不能用来分红

例子：宁波银行2024年净利润200亿，必须拿20亿放进盈余公积这个"存钱罐"里锁起来，不能全部分给股东。
4. 未分配利润（留存收益）
通俗说法：累计没分红的利润（银行赚的钱剩下的部分）
计算公式：
期初未分配利润+ 本年净利润 - 提取盈余公积（10%） - 分给股东的现金分红 - 一般风险准备= 期末未分配利润

这是银行历年积攒的家底
可以分红，但分多了核充率就下降
银行通过"少分红、多留存"来提升核充率

例子:招行2024年净利润1500亿，分红30%（450亿），剩下的1050亿里：150亿进盈余公积（10%）900亿留在未分配利润里，增加核充

5. 一般风险准备
通俗说法：专门应对坏账的"备用金"（从利润里计提）
与拨备的区别：
拨备覆盖率：针对已发生的不良贷款（进利润表，抵减利润）
一般风险准备：针对未来可能发生的损失（进资产负债表，作为资本）
监管要求：
不得低于风险加权资产的1.5%
从净利润里提取，不能分红

例子：
如果宁波银行RWA是1万亿，那么一般风险准备至少要有150亿。这部分钱趴在账上不动，专门等 crisis 时用
分母部分：风险加权资产（RWA）
通俗说法：银行放出去的"风险贷款"按权重折算后的总额
为什么叫"风险加权"：
不是简单看放了多少贷款，而是看这些贷款有多危险
例子：
宁波银行放出两笔贷款各100亿：
一笔给政府平台（风险权重20%）→ RWA = 20亿
一笔给民营企业（风险权重100%）→ RWA = 100亿
虽然都是100亿，但后者消耗的资本是前者的5倍！
大白话解释就是
银行股东投的钱（普通股+溢价）+ 历年攒下的利润（盈余公积+未分配利润）+ 风险备用金（一般风险准备）÷ 放出去的钱按风险折算后的总额

| ------------ | ------ | ------------- |
| 资产类型         | 风险权重   | 解释            |
| 国债           | 0%     | 国家不会违约，无风险    |
| 政策性金融债       | 0%     | 国家信用背书        |

2023年：RWA=18000亿，一般风险准备=270亿（正好1.5%） 2024年：RWA增长15%至20700亿，需要的一般风险准备=310亿因此2024年必须从利润中**额外提取40亿**（310-270）来补足


第二个指标 **拨备覆盖率**
这是银行"粮草储备"的指标，反映银行对坏账的"御寒能力"
拨备覆盖率=银行攒的坏账准备金/实际已经形成的坏账*100%
果银行有100亿坏账收不回来了，账上准备了300亿"备胎钱"，那么拨备覆盖率就是**300%**
银行从每年利润中提前计提的、专门用来核销坏账的"专项公积金"。
不是现金！不是现金！不是现金！  （重要说三遍，这是会计上的"准备"，是权益的抵减项）
每年银行做利润表时：

营业收入- 利息支出- 管理费用- 税金及附加- **信用减值损失（就是拨备）** ← 这里计提= 利润总额

宁波银行2024年利润200亿，其中拿出50亿作为"信用减值损失"（计提拨备），这50亿就进了"贷款损失准备"这个池子里

### **三阶段计提模型（IFRS 9）**

### **银行根据贷款风险程度，分三种情况提钱**
|   |   |   |   |
|---|---|---|---|
|阶段|风险程度|计提标准|例子|
|阶段一|正常|未来12个月可能违约的金额|普通企业贷款，经营正常|
|阶段二|关注|整个贷款周期可能违约的金额|企业有点困难，但还在付息|
|阶段三|不良|整个贷款周期违约金额（100%或部分）|企业已经破产或逾期90天+|

银行把贷款按风险分成5档，后3档合称"不良贷款"：

|     |              |                    |
| --- | ------------ | ------------------ |
| 分类  | 定义           | 通俗解释               |
| 正常  | 借款人能按时还本付息   | 好贷款                |
| 关注  | 目前正常，但存在潜在风险 | 看着有点悬，需要盯紧         |
| 次级  | 还款能力明显出问题    | 已经逾期或肯定还不上，但还有点抵押物 |
| 可疑  | 肯定还不上了       | 抵押物拍卖也不够还          |
| 损失  | 基本没戏了        | 钱打水漂了，只是在等走法律程序核销  |
不良贷款 = 次级 + 可疑 + 损失   不良率=不良贷款/贷款总额×100% 
**实际案例：宁波银行 vs 招商银行**

假设两家银行贷款总额都是**1万亿元**

|   |   |   |   |
|---|---|---|---|
|指标|宁波银行|招商银行|差异分析|
|不良率|0.76%|0.94%|宁波表面更优|
|不良贷款额|76亿|94亿|招行坏账多18亿|
|拨备覆盖率|373%|392%|招行准备更充足|
|拨备总额|283亿|368亿|招行多准备85亿|
|隐藏利润|多|更多|都超过监管要求150%|
**计算过程：**

**宁波：76亿 × 373% = 283亿拨备**
**招行：94亿 × 392% = 368亿拨备**
**监管要求演变**

|   |   |   |
|---|---|---|
|时间|监管要求|政策意图|
|2000s|100%|基本覆盖|
|2011|150%|逆周期调节，要求多留粮|
|2018|120%-150%|差异化监管，好银行可以低一点|
|2020-2022|阶段性下调|疫情让银行多放贷，允许拨备降一点|
|2024至今|150%|重新收紧，要求做实资产质量|
当前红线：≥ 150%（多数银行执行）

为什么不能太低？

低于150%：监管认为你"准备不足"，如果经济下行坏账爆发，银行会亏损甚至破产

低于120%：会被监管约谈，限制分红、限制扩张

为什么不能太高？（藏利润）

监管隐忧上限：400%

逻辑：

监管只要求150%，如果银行做到400%，多出来的250%就是"超额拨备"

超额拨备 = 隐藏利润

藏利润的操作手法：

复制

今年利润很好（比如200亿）

→ 多提拨备（比如提80亿，实际需要只有50亿）

→ 账面利润变成120亿（少交企业所得税）

→ 多提的30亿趴在拨备池子里

→ 明年利润不好时，转回30亿变成利润

典型案例：
招商银行：拨备覆盖率392%，市场一直认为其"藏了利润"（可以释放约200亿利润）
宁波银行：373%，同样存在藏利润空间
杭州银行：550%+，藏得最多，所以市场给其估值溢价（认为未来利润有缓冲）

## **拨备覆盖率与核充率的区别**

|       |                |                      |
| ----- | -------------- | -------------------- |
| 维度    | 拨备覆盖率          | 核充率（核心一级资本充足率）       |
| 性质    | 针对贷款的风险准备      | 针对所有资产的资本缓冲          |
| 位置    | 资产负债表负债端（抵减贷款） | 资产负债表权益端（股东权益）       |
| 来源    | 从利润中计提         | 股东投入+利润留存            |
| 用途    | 核销具体坏账         | 吸收所有损失（包括非贷款损失）      |
| 监管红线  | ≥150%          | ≥7.5%（最低）/ ≥9.5%（实际） |
| 与分红关系 | 拨备太低不能分红       | 核充太低不能分红             |

关键联系：

核销坏账时：先用拨备，不足再用利润，最后才侵蚀资本
所以拨备是第一道防线，核充是最后一道防线
�� 投资者怎么看？
健康区间
150%-250%：正常，符合监管，利润释放空间适中
250%-400%：优秀，有隐藏利润，未来业绩有缓冲
>400%：可能存在过度审慎或藏利润，要看看是不是不良认定太严
<150%：危险，可能面临业绩暴雷或被迫大补拨备导致利润下滑

警惕信号
拨备覆盖率快速下降（如从300%降至200%）：可能经济下行，坏账增加
拨备覆盖率<150%且不良率上升：双杀，赶紧跑
拨备覆盖率>500%但不良率<0.5%：可能不良认定过严（好事），也可能在藏利润（需要分析）
一句话总结
拨备覆盖率是银行的"存粮"，150%是过冬底线，300%是富裕线，超过400%就是"粮仓里的老鼠"（藏利润）了。
**ROE（净资产收益率）**是银行股东最关心的指标——**"我投进去的100块钱，一年能赚回多少"**。

对于银行这种高杠杆行业，ROE的分析有特殊技巧
ROE=净利润 /净资产（股东权益）*100%
银行ROE的"及格线"：
≥15%：优秀（如招商银行历史上的表现）
12%-15%：良好（多数好银行在这个区间）
8%-12%：及格（能跑赢通胀，但吸引力下降）
<8%：不及格（不如买国债，资本充足率压力也会显现）

2025年行业现实：多数银行ROE已跌至10-12%，招行也从曾经的17%跌至13-14%
银行ROE要用改进版杜邦分析（考虑银行特殊性）：
ROE=权益乘数×资产收益率（ROA）
进一步拆解ROA：
ROE=权益乘数×(净息差−信用成本率−成本收入比−税负率)

#### **① 资产回报率 (ROA)：钱赚得辛苦吗？**

这是衡量银行管理水平的核心。主要看三点：
**净息差 (NIM)：** 这是银行的“剪刀差”。例如成都银行和江苏银行在 2025 年通过“以量补价”维持了利润增长。
**非利息收入占比：** 优秀的银行（如宁波银行）手续费及佣金收入增长较快，这种业务不消耗资本，是推高 ROE 的“高级燃料”。
**成本收入比：** 银行每赚 100 块钱要花掉多少运营成本。
#### **② 权益乘数：杠杆加得高吗？**
银行本质是杠杆生意。但杠杆不能乱加，受**核心一级资本充足率**的严格监管。
**分析点：** 如果一家银行 ROE 很高，但核充率逼近监管红线（如第一组 D-SIBs 的 7.75%），说明它的高 ROE 是靠把杠杆加到极致换来的，未来增长空间受限。

### **4. 2026 年的实战建议**

在当前“息差企稳”的预期下，分析 ROE 时应重点关注：

**分红稳定性：** 赚的钱愿不愿意分？**资产质量底色：** 不良率是否保持在低位（如 0.8% 以下）？ 因为坏账是 ROE 的“粉碎机”。**资本效率：** 观察银行是否能以更小的资本消耗（较低的资产增速）换取更高的利润增长。例如 2025 年部分优秀城商行利润增速保持在 8%-10% 左右，显示了较强的韧性。

|<9.5%|危险（<1%）|⭐⭐|平安银行（9.3%）|
|<8.5%|枯竭|⭐|必须零分红保资本|
**关键信号**：
**核充率连续下降**（如宁波银行从11%→9.4%）→ 分红不可持续
**RWA增速>核充内生增速**（即利润留存速度）→ 未来必须降分红或再融资

### **维度2：利润质量（是真钱还是纸面富贵）**

**警惕"假利润"**： 银行可以通过**少提拨备**做高利润，但不可持续，最终要回补，导致未来分红下降
拨备覆盖率趋势
稳定或上升（好）：利润真实，分红可持续
快速下降（坏）：如从400%→250%，可能在吃老本，未来要补提
关注类贷款占比
关注类是不良的"蓄水池"，若从0.5%→1.0%，未来不良爆发会侵蚀利润
ROE vs 分红率匹配
合理区间：ROE×(1−分红率)≈RWA增速
若ROE 12%，分红50%，留存仅6%，但RWA增长15%，缺口9%必须融资，分红不可持续

| --- | ------ | --- | ------ | ----- | ------- | ------------ |
| 银行  | ROE    | 分红率 | 留存收益增速 | RWA增速 | 匹配度     | 分红可持续性       |
| 招行  | 13.44% | 33% | 9.00%  | 8%    | ✅       | 可持续          |
| 宁波  | 13.10% | 30% | 9.20%  | 16%   | ❌缺口6.8% | 不可持续，需降分红或融资 |
| 常熟  | 12%    | 25% | 9.00%  | 12%   | ⚠️缺口3%  | 偏紧，看核充率      |

**维度3：股东结构与分红意愿（愿不愿分）**
**不同类型银行的分红动机**：


|   |   |   |   |   |
|---|---|---|---|---|
|类型|大股东|分红意愿|稳定性|特点|
|国有大行|财政部/汇金|⭐⭐⭐⭐⭐|极高|30%是政治任务，且要"多次分红"响应国九条|
|股份行|分散/民营|⭐⭐⭐|中等|看董事会决策，如招行稳定，平安波动|
|城商行|地方财政|⭐⭐⭐⭐|较高|地方财政需要分红收入，但受限于资本|
|农商行|地方/民营|⭐⭐|较低|经营波动大，如渝农商行股息率高但不稳定|
特殊信号：
大股东缺钱：如地方政府财政紧张，可能要求城商行提高分红（即使核充紧张）
再融资前：银行会提高分红"讨好"老股东，以便后续配股（如宁波银行2023年配股前承诺40%分红）

### **维度4：历史记录与承诺（敢不敢分）**

1. **看历史分红率稳定性**

理想状态：过去5年分红率稳定在25%-35%，波动<5个百分点
危险信号：分红率从40%→30%→25%（逐年下降，利润或资本出问题）
2. **看分红承诺**
宁波银行：承诺2024-2026年"不低于30%"（有明确承诺的好于没承诺的）
招行：无明确数字承诺，但"保持合理稳定"（实际33%左右）
四大行：无明确数字，但实际稳定在30%

3. **看分红频次**
一年一次：传统，现金流不确定
一年两次（中期+末期）：如招行、工行已实施，确定性更强（中期分红一般在9-10月，可提前锁定）

**分红"暴雷"的5个前兆信号**
信号1：核充率跌破10%且RWA增速>12%
后果：6-12个月内必须降分红或融资
当前案例：宁波银行（9.4%核充+16%RWA增速）、南京银行（9.31%）
信号2：拨备覆盖率<200%且不良率上升
后果：利润要被拿去补拨备，没钱分红
危险银行：浦发银行（拨备170%）、华夏银行
信号3：关注类贷款占比>1.5%
后果：未来1-2年不良爆发，侵蚀利润
监测：财报附注"贷款五级分类"中"关注类"数据
信号4：前一年大额再融资（配股/定增）
逻辑：刚圈了钱，如果又高分红，等于"把股东的钱分红给股东"（耍流氓）
案例：宁波银行2023年配股120亿，2024年分红70亿，实际上是用新股东的钱给老股东分红
信号5：营收负增长但利润正增长
猫腻：通过少提拨备做高利润，不可持续
当前案例：中信银行（2025年营收-0.55%，利润+2%），未来分红有压力
**分红稳定性评估公式**：
分红确定性=核充率−RWA增速+拨备覆盖率趋势+国有背景系数
分红确定实操口诀：
核充>12% + 分红<35%：稳如老狗（放心买）
核充<10% + 分红>30%：击鼓传花（快逃）
营收负增 + 利润正增：假分红（别碰）
刚配股 + 高分红：套路（别信）
**当前（2025年）最稳分红标的**：**四大行（工建农中）+ 招商银行**，其他银行的"高分红"承诺都需打问号。
