# 基本面投资减法框架（曼巴投资法）

> 来源：雪球曼巴投资  
> 核心思想：做减法、差异化、80/20法则  
> 整理日期：2026-03-12  
> 适用：基本面价值投资、能力圈构建

---

## 一、投资的本质（最简版本）

### 1.1 核心不等式
```
价格 < 价值
```

巴菲特1984年演讲：格雷厄姆和多德投资者**只考虑这一点**。

### 1.2 价值的本质
- **价格是客观的**：每个当下的价格可观测
- **价值是主观的**：未来现金流是预测的，必然是主观
- **方法是客观的**：DCF公式是确定的

**结论**：抛开具体公司，无法讨论价值投资怎么落地。

## 二、筛选第一性：差异化（垄断竞争）

### 2.1 经济学原理
```
完全竞争 → 零利润（仅覆盖成本，ROE≈无风险利率）
垄断竞争 → 超额利润（差异化带来定价权）
```

### 2.2 投资筛选标准
问自己三个问题：
1. **能不能提供差异化的产品和服务？**
2. **能不能赚到肉眼可观的利润？**
3. **能不能持续好长的一段时间？**

其实就是对着**DCF公式**一个个检查：
- 赚得多（利润率高）
- 赚得久（可持续性强）
- 赚得容易（竞争优势明显）
- +自己看得懂
- +估值一眼可见的低估

**这些条件同时满足时，往往是市场恐慌时**——"别人恐慌的时候我贪婪"，反而是最简单的。

## 三、竞争优势（护城河）类型

### 3.1 六大竞争优势

| 类型 | 代表公司 | 量化方法 |
|------|---------|---------|
| **品牌** | 茅台、泡泡玛特 | 毛利率-销售费用率（用户粘性） |
| **网络效应** | 微信、滴滴 | 用户数、双边平台规模 |
| **规模效应** | 美团、名创优品 | 单均成本随规模下降 |
| **成本优势** | 拼多多、Costco | 净利润/GMV效率 |
| **供应链优势** | SHEIN、Zara | 库存周转、快反能力 |
| **技术/专利** | 华为、宁德时代 | 研发投入产出比 |

### 3.2 作者能力圈聚焦
**个人专注三类（可数学化/结构化）**：

1. **品牌**：毛利率-销售费用率及历史变化、同行比较
2. **网络效应**：极致的规模效应，边际成本为0甚至负
3. **规模效应**：成本优势的本质

**对应标的类型**：
- 消费品股（品牌驱动）
- 互联网股（网络效应、规模效应）
- 其他：银行、保险、物业、地产等数据驱动业务

## 四、研究的80/20法则（核心方法论）

### 4.1 四步研究法

```
Step 1: 数据化（20%时间抓80%）
    ↓
Step 2: 找核心变量
    ↓
Step 3: 对比老二（4倍法则）
    ↓
Step 4: 看天花板
```

### 4.2 详细步骤

#### ① 资产负债表 + 利润表（数据化）

**资产负债表关键指标**（全部用/股东权益标准化）：

| 指标 | 公式 | 意义 |
|------|------|------|
| 净负债率 | (有息负债-现金)/股东权益 | 经营靠现金还是借钱 |
| 重资产占比 | (固定资产+无形资产+土地)/股东权益 | 轻资产 vs 重资产 |
| 上下游占款 | (应付款-应收款)/股东权益 | 产业链地位 |
| 存货占比 | 存货/股东权益 | 库存风险 |
| 商誉占比 | 商誉/股东权益 | 内生增长 vs 并购扩张 |

**利润表关键指标**：
- 历史收入/利润增长
- 净利润率（盈利能力）
- 毛利率（用户离开的成本）
- 销售费用率（获取用户的成本）
- 四费率（销售/管理/财务/研发）

**毛利率-销售费用率 = 产品粘性**

#### ② 产品和业务 + 数据化 → 找核心变量

结合日常常识，找到经营需要关注的最核心变量：

| 公司 | 核心变量 | 观察指标 |
|------|---------|---------|
| **滴滴** | 双边规模效应 | 日均订单量、行业份额 |
| **美团** | 即时配送网络 | 即时订单量、每单利润 |
| **亚朵** | 场景零售转型 | 电商销售占比、房间数、RevPAR |
| **腾讯** | 网络效应 | 微信用户数、游戏/金融/云增速 |
| **海底捞** | 品牌+服务 | 翻台率、单价、毛利率-费用率 |
| **泡泡玛特** | 品牌+稀缺性 | 二手市场份额、门店数 |
| **名创优品** | 规模效应 | 门店数、同店增长、老二差距 |

#### ③ 和老二的差别（4倍法则）

**关键标准**：老大 ≈ 4个老二，即老大占80%以上份额

| 公司 | 老二情况 | 差距 |
|------|---------|------|
| 泡泡玛特 | 二手潮玩TOP20中18个是泡泡玛特 | 碾压 |
| 名创优品 | 同模式门店老二少很多且未必赚钱 | 领先 |
| 微信 | 支付宝（不同赛道）、其他社交软件 | 绝对优势 |

**例外**：PDD当年逆袭阿里，就是效率更高逆袭了"4倍"

#### ④ 行业天花板

**最好肉眼可见还有很大空间**：

| 公司 | 天花板情况 |
|------|-----------|
| 亚朵 | 20万间 vs 行业2000万间（1%），空间巨大 |
| 泡泡玛特 | 门店数刚开始，海外刚开始 |
| 名创优品 | 门店数已较多，但海外还有空间 |

**估值角度**：静态特别低估时，天花板要求可以降低。

#### ⑤ 颠覆风险（硬塞进来）

**产品服务线性推演多年不会被颠覆**：

- 出租车 → 无人驾驶？（需要观察跟踪）
- 白酒 → 被替代？（文化传承，难替代）
- 手办 → 数字藏品替代？（情感连接，难替代）

## 五、时间分配哲学

### 5.1 80/20法则的量化

```
20%时间 → 掌握80%（广度）
100%时间 → 掌握100%（深度）

效率比 = 16倍

选择：
- 100%时间搞懂1家公司100% → 单调、集中风险
- 20%时间搞懂16家公司80% → 组合、风险分散、收益可能更高
```

### 5.2 深度 vs 细度

| 维度 | 定义 | 时间投入 |
|------|------|---------|
| **深度** | 第一性原理的理解 | 20%时间足够 |
| **细度** | 细节和技术的汪洋大海 | 80%时间沉没 |

**建议**：沉迷细度不一定是深度，可能陷入细节无法自拔。

## 六、数据处理注意事项

### 6.1 一次性影响要剔除

**案例**：泡泡玛特2022年
- 表面：收入增长仅3%
- 实际：疫情、线下关闭影响
- 核心：Molly数据没下滑
- 结论：不能简单说"潮玩手办完了"

### 6.2 数据会说话

**案例**：拼多多早期
- 表面：烧钱、亏损
- 实际：净利润/GMV < 1%（每1元GMV亏不到1分钱）
- 结论：不是真烧钱，效率极高
- 对比：哔哩哔哩（真烧钱，不认可）

## 七、财务分析实操（万得/东财）

### 7.1 三张表导出
- 用万得或东财专业版导出Excel
- 时间序列数据（能导出多少年就多少年）

### 7.2 处理顺序

**资产负债表**（核心）→ **利润表**（核心）→ **现金流量表**（验证）

### 7.3 现金流表推导

资产负债表 + 利润表 → 可以推导出现金流量表

所以前两个功课做好，现金流表就是验证。

### 7.4 行业比较注意

**财务指标不要轻易跨行业比较**：

| 行业 | 特点 | 现金流特征 |
|------|------|-----------|
| 重资产（三峡、长电） | 折旧大 | 经营现金流 >> 净利润 |
| 轻资产（茅台） | 折旧小 | 经营现金流 ≈ 净利润 |

## 八、总结：减法框架 checklist

### 投资前筛选（做减法）

- [ ] 价格 < 价值？（一眼可见的低估）
- [ ] 有差异化/竞争优势？（非完全竞争）
- [ ] 品牌/网络/规模效应？（可数学化验证）
- [ ] 和老二差距4倍以上？（竞争格局）
- [ ] 天花板还很高？（成长空间）
- [ ] 不容易被颠覆？（持续性）
- [ ] 自己看得懂？（能力圈）

### 研究时聚焦（80/20法则）

- [ ] 资产负债表数据化（5大比率）
- [ ] 利润表数据化（4大费率）
- [ ] 找到核心经营变量
- [ ] 对比老二差距
- [ ] 评估天花板高度
- [ ] 识别一次性影响
