## 一、龙虎榜数据 - 有必要，但有局限

### 1.1 龙虎榜能看到什么？

| 信息 | 作用 | 局限 |
|------|------|------|
| **营业部买卖金额** | 看出主力/游资进出 | 只能看当日，无法看历史连续 |
| **机构席位** | 判断机构是否在买 | 机构可能分多日建仓，单日数据不完整 |
| **买入/卖出前五** | 判断资金态度 | 游资一日游还是机构锁仓 |

### 1.2 使用建议

- ✅ **涨停/跌停当天**：必须看，判断是谁在买（机构vs游资）
- ✅ **连续异动**：连续上榜说明有持续资金关注
- ⚠️ **滞后性**：收盘后才公布，只能验证不能预测
- ❌ **不要迷信**：游资今天买明天可能砸盘

## 二、K线图 - 必须关注

### 2.1 K线能提供的信息

| 维度 | 内容 | 重要性 |
|------|------|--------|
| **价格趋势** | 上涨/下跌/横盘 | ⭐⭐⭐⭐⭐ |
| **成交量** | 放量/缩量/天量 | ⭐⭐⭐⭐⭐ |
| **支撑阻力** | 前期高低点、均线 | ⭐⭐⭐⭐ |
| **形态** | 头肩顶、双底等 | ⭐⭐⭐ |
| **资金流向** | 量价配合/背离 | ⭐⭐⭐⭐ |

---

## 三、什么情况下需要截图K线？

### 3.1 建议截图的场景

| 场景 | 说明 |
|------|------|
| **技术分析为主** | 要判断买卖点、支撑阻力 |
| **刚涨停/跌停** | 要分析资金意图 |
| **形态判断** | 如"是不是双顶"、"有没有突破" |
| **多周期对比** | 日线+周线+月线一起看 |

### 3.2 不需要截图的场景

| 场景 | 说明 |
|------|------|
| **基本面分析为主** | 财报、行业分析，K线不重要 |
| **估值判断** | PE/PB分析，股价数字即可 |
| **大师框架分析** | 巴菲特、芒格框架，不依赖技术 |

## 四、完整分析流程（结合K线）

```
用户给股票代码 + 可选截图K线
         ↓
    查持仓记录（是否已持有）
         ↓
    财报评分（基本面）
         ↓
    大师框架分析（价值判断）
         ↓
    【如果有K线图】→ 技术分析
         ↓
    综合建议
```

## 五、数据源可信度分级

| 数据类型 | 来源 | 可信度 |
|----------|------|--------|
| **K线图** | 雪球/同花顺截图 | ⭐⭐⭐⭐ 高（市场数据） |
| **龙虎榜** | 交易所公布 | ⭐⭐⭐⭐⭐ 最高（官方数据） |
| **成交量/换手率** | 行情软件 | ⭐⭐⭐⭐ 高 |
| **资金流入流出** | 软件估算（如"主力净流入"）| ⭐⭐ 低（算法估算，非真实）|

> ⚠️ **重要提醒**：软件显示的"主力资金流入"是**估算值**，不可信。真实的只有龙虎榜公布的营业部买卖数据。

## 七、实操建议

### 给投资者的建议

1. **不要只看K线做决策** - 结合基本面和估值
2. **龙虎榜只是参考** - 不要迷信"主力"，主力也会亏钱
3. **量价关系最重要** - 放量上涨是真突破，缩量上涨要警惕
4. **截图要全面** - 至少包含日K线+成交量+关键指标（MACD/KDJ）

### 给AI分析的建议

- 如果是**技术分析为主**（如"这股能不能追"、"支撑在哪"）→ **截图K线**
- 如果是**基本面分析**（如"这公司好不好"）→ **不用截图**，给代码即可
